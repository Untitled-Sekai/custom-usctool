# USCTool

USCTool は、Versioned Universal Sekai Chart を扱うためのツールです。
JS ライブラリ、CLI ツール、dll ライブラリの 3 つの形態で提供されます。

## インストール

### JS ライブラリ

```bash
npm install custom-usctool
```

### CLI ツール / dll ライブラリ

[リリースページ](https://github.com/Untitled-Sekai/custom-usctool/releases) からダウンロードしてください。
